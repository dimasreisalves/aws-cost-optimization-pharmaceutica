# Implementação de Redução de Custos AWS - Abstergo Industries

## 🚀 Descrição do Projeto
Este projeto foi desenvolvido como parte do curso na DIO, com o objetivo de criar um plano estratégico de migração e otimização de custos para a empresa farmacêutica fictícia **Abstergo Industries**. O foco principal é reduzir despesas operacionais (OpEx) utilizando serviços gerenciados da AWS.

## 🛠️ Tecnologias Utilizadas
* **AWS EC2 Spot Instances**: Para processamento de dados de P&D.
* **Amazon S3 Intelligent-Tiering**: Para gestão eficiente de armazenamento de dados regulatórios.
* **AWS Cost Explorer**: Para governança financeira e monitoramento.
