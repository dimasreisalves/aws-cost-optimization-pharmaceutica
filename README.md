# aws-cost-optimization-pharmaceutica
Projeto desenvolvido para o desafio da DIO focado em Cloud Economics. Apresento uma solução para a Abstergo Industries utilizando Amazon EC2 Spot Instances para redução de custos em P&amp;D, S3 Intelligent-Tiering para gestão de dados regulatórios e AWS Cost Explorer para FinOps. O relatório completo e estruturado está disponível no repositório GitHub.
